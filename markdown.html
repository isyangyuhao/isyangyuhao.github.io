零. 概述

  1. Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式
  2. 不在 Markdown 涵盖范围之内的标签，都可以直接在文档里面用 HTML 撰写。不需要额外标注这是 HTML 或是 Markdown；只要直接加标签就可以了

一. 标题

  1. Setext形式标题写法 
    这是一级标题(在标题下方的等号数量无限制)
=============
这是二级标题(在标题下方的中横线数量无限制)
-------------
  2. Atx形式的标题写法 
    # 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题
  3. Atx形式的标题写法中,可以选择性的闭合标记,闭合标记纯粹是处于美观,并无语法要求且闭合的井号个数无限制 
    # 这是闭合的一级标题 #
## 这是闭合的二级标题 ##
##### 这是闭合的五级标题,闭合的井号只写一个也没问题 #

三. 列表

  1. 无序列表
    • 无序列表可以使用星号作为标记 
      * 无序列表1
* 无序列表2
* 无序列表3
    • 无序列表可以使用加号作为标记 
      + 无序列表1
+ 无序列表2
+ 无序列表3
    • 无序列表可以使用减号作为标记 
      - 无序列表1
- 无序列表2
- 无序列表3
  2. 有序列表
    有序列表使用数字接着一个英文句点 
    1. 有序列表1
2. 有序列表2
3. 有序列表3

四. 代码区块

  1. 使用四个缩进或一个制表符将会转换为代码区块 
        System.out.println("hello world!");
  2. 需要标记一段行内代码,可以使用反引号将其包裹起来 
    this is code: `using namespace;`
  3. 如果需要在代码片段中使用反引号,则需要使用双反引号标注行内代码 
    this is code: `` i want use `  ``

五. 分割线

  1. 使用三个以上的星号、减号来建立分割线,行内不能存在其他内容 
    *********************
  2. 分割线中间允许插入空格 
    * * * * * * * * * * *
