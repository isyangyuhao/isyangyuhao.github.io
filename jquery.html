1. jQuery对象

  1. jQuery对象
    • jQuery对象就是通过jQuery($())包装DOM对象后产生的对象
    • jQuery对象是jQuery独有的，如果一个对象是jQuery对象，那么就可以使用jQuery中的方法
    • jQuery对象无法使用DOM对象中的任何方法，同样DOM对象也无法使用jQuery对象的任何方法
    • 约定：如果获取的对象是jQuery对象，则在变量前加上"$"
      var obj = document.getElementById("objIdName");
      var $obj = $("#objIdName");
  2. jQuery对象转换为DOM对象
    • jQuery对象是一个数组对象，可以通过[index]方法得到对应的DOM对象
      var $obj = $("#objIdName");
      var obj = $obj[0];
    • 使用jQuery的get(index)方法得到对应的DOM对象
      var $obj = $("#objIdName");
      var obj = $obj.get(0);
  3. DOM对象转换为jQuery对象
    • 使用$()包装DOM对象
      var obj = document.getElementById("objIdName");
      var $obj = $(obj);

2. 选择器

  • 选择器是jQuery的根基，在jQuery中，对事件的处理，遍历DOM，操作Ajax都依赖选择器
  • 选择器的优点: 写法简洁, 完善的事件处理机制
  1. 基本选择器
    基本选择器是jQuery中最常用的选择器，通过元素的id,class,标签名查找DOM元素
    选择器 描述 返回 e.g.
    $("#id") 根据给定的id匹配一个元素 单个元素组成的集合 $("#content")
    $(".class") 根据给定类名匹配元素 集合元素 $(".title")
    $("element") 根据给定的元素名匹配元素 集合元素 $("div")
    $("*") 匹配所有元素 集合元素 $("*")
    $("selector1, selector2, selectorN") 将每一个选择器匹配到的元素合并后一起返回 集合元素 $("#content, .title")
  2. 层次选择器
    通过DOM元素之间的层次关系来获取特定元素，例如后代元素，子元素，相邻元素，兄弟元素等
    选择器 描述 返回 e.g.
    $("ancestor descendant") 选取ancestor的所有descendant(后代)元素 集合元素 $$("div span")
    $("parent > child") 选取parent元素下的child(子)元素 集合元素 $("#bigDiv div")
    $("prev + next") 选取紧接在prev元素后的next(下一个)元素 集合元素 $(#bigDiv div)
    $("prev ~ siblings") 选取prev元素后的同辈的全部siblings元素 集合元素 $(#content ~ div)
    $(obj1).siblings(obj2Name) 选取obj1元素同辈的全部obj2Name元素 集合元素 $("#content").siblings("div")
    $(obj1).prevAll(obj2Name) 选取obj1元素前的同辈的全部objName2元素 集合元素 $("#content").prevAll("div")
    $(obj1).nextAll(obj2Name) 选取obj1元素后的同辈的全部obj2Name元素 集合元素 $("#content").nextAll("div")
  3. 过滤选择器
    • 过滤选择器主要是通过特定的过滤规则来筛选出所需的DOM元素，该选择器以":"开头
    • 基本过滤选择器
      选择器 描述 返回 e.g.
      :first 选取第一个元素 单个元素组成的集合 $("div:first")
      :list 选取最后一个元素 集合元素 $("div:last")
      :not(selector) 去除所有给定选择器匹配的元素 集合元素 $("div:not(.one)")
      :even 选取索引是偶数的所有元素，索引从0开始 集合元素 $("div:even")
      :odd 选取索引是奇数的所有元素，索引从0开始 集合元素 $("div:odd")
      :eq(index) 选取索引等于index的所有元素，索引从0开始 集合元素 $("div:gt(3)")
      :gt(index) 选取索引大于index的所有元素，索引从0开始 集合元素 $("div:lt(3)")
      :lt(index) 选取索引小于index的所有元素，索引从0开始 集合元素 $("div:lt(3)")
      :header 选取所有标题元素，如h1, h2等 集合元素 $(":head")
      :animated 选取正在执行动画的元素 集合元素 $(":animated")
    • 内容过滤选择器 - 体现在所包含的子元素和文本内容上
      选择器 描述 返回 e.g.
      :contains(text) 选取含有文本内容为text的元素 集合元素 $("div:contains('di')")
      :empty 选取不包含子元素或文本的空元素 集合元素 $("div:empty")
      :has(selector) 选取含有选择器所匹配的元素的元素 集合元素 $("div:has(.mini)")
      :parent 选取含有子元素或文本的元素 集合元素 $("div:parent")
    • 可见性过滤选择器 - 根据元素的可见和不可见状态(包含display:none&隐藏域等)来选择对应元素
      选择器 描述 返回 e.g.
      :hidden 选取所有不可见元素 集合元素 $("div:hidden")
      :visible 选取所有可见的元素 集合元素 $("div:visible")
    • 属性过滤选择器 - 通过元素的属性来获取相应的元素
      选择器 描述 返回 e.g.
      [attribute] 选取拥有此属性的元素 集合元素 $("div[title]")
      [attribute=value] 选取指定属性的值为value的元素 集合元素 $("div[title='test']")
      [attribute!=value] 选取指定属性的值不为value的元素 集合元素 $("div[title!='test']")
      [attribute^=value] 选取指定属性以value开始的元素 集合元素 $("div[title^='te']")
      [attribute$=value] 选取指定属性以value结束的元素 集合元素 $("div[title$='st']")
      [attribute*=value] 选取指定属性含有value的元素 集合元素 $("div[title*='es']")
      [selector1][selector2]…[selectorN] 用属性选择器合并成一个复合属性选择器，
      满足多个条件，每选择一次缩小一次范围      		 集合元素 $("div[title^='te'][title$='st']")
    • 子元素过滤选择器 (选择子元素必须在选择器前加一个空格)
      选择器 描述 返回 e.g.
      :nth:child(index/
      even/odd/equation)      		 选取每个父元素下第index个子元素 或者奇偶元素(index从1算起) 集合元素 $("div: nth:child(3)")
      :first-child 选取每个父元素的第一个子元素 集合元素 $("div: first-child")
      :last-child 选取每个父元素的最后一个子元素 集合元素 $("div: last-child")
      :only-child 如果某个元素是它父元素中的唯一子元素，那么将会被匹配 集合元素 $("div: only-child")
    • 表单对象属性过滤选择器 - 对所选择的表单进行过滤
      选择器 描述 返回 e.g.
      :enabled 选取可用的元素 集合元素 $(":text:enabled")
      :disabled 选取不可用的元素 集合元素 $(":text:disabled")
      :checked 选取所有被选中的元素(radio,checkbox) 集合元素 $(":checkbox[name='t']:checked")
      :selected 选取所有被选中选项的元素(下拉列表) 集合元素 $("select: selected")
      [option是select子元素,需要打空格]
      [通过each()方法可以遍历得到DOM对象]
  4. 表单选择器
    选择器 描述 返回
    :input 选取<input><textarea><select><button>等元素 集合元素
    :text 选取所有单行文本框 集合元素
    :password 选取所有密码框 集合元素
    :radio 选取所有单选框 集合元素
    :checkbox 选取所有多选框 集合元素
    :submit 选取所有提交按钮 集合元素
    :image 选取所有图像按钮 集合元素
    :reset 选取所有重置按钮 集合元素
    :button 选取所有按钮 集合元素
    :file 选取所有上传域 集合元素
    :hidden 选取所有不可见元素 集合元素

3. DOM操作

  1. DOM操作的分类
    • DOM Core: DOM Core并不属于Javascript, 任何一种支持DOM程序设计语言都可以使用，他的用途非仅限于处理网页，可以处理任何文本标记语言，例如XML
    • HTML DOM: 使用Javascript和DOM为HTML文件编写脚本时，有许多专属HTML-DOM属性
    • CSS-DOM: 针对CSS操作，在Javascript中，CSS-DOM主要用于获取和设置style对象的各种属性
  2. 查找结点
    • 查找元素结点：通过jQuery选择器完成
      e.g. 获取输入用户名的单行文本框
      var $inputUsername = $(":text[name='username']");
    • 操作属性结点：查找到所需要的元素后通过jQuery对象的attr()方法来获取并操作各种属性值
      e.g.1 读取文本框中的value值
      alert($inputUsername.attr("value"));
      e.g.2 修改文本框中的value值
      $inputUsername.attr("value", "testAttr");
    • 操作文本结点：通过text()方法操作文本节点
      e.g.1 读取文本节点
      alert($("#span").text())
      e.g.2 修改文本节点
      $("#span").text("testText");
  3. 创建节点
    • 使用jQuery的工厂方法$() : $(html)会根据传入的html标记字符串创建一个DOM对象，并将一个DOM对象包装成一个jQuery对象返回
    • 动态创建的新元素节点不会自动添加到文档中，而是需要其他方法将其插入到文档之中
    • 当创建单个元素的时候，需要注意闭合标签和使用标准的XHTML格式，例如创建p元素节点：$("<p/>")或$("<p></p>")可以创建，不可以使用$("<p>")或者$("<P>")
    • 创建文本节点就是在创建元素节点的时候直接将文本内容写出来，创建属性节点也是在创建元素节点时一起创建
  4. 插入节点
    • 动态创建HTML元素并没有实际作用，需要将新创建的节点动态插入到文档中，即成为某个元素的子节点
      方法 描述
      append() 向每个匹配的元素的内部结尾处追加内容
      appendTo() 将每个匹配的元素追加到指定元素的内部的结尾处
      prepend() 向每个匹配的元素的内部的开始处插入内容
      prependTo() 向每个匹配的元素插入到指定元素的内部的开始处
      after() 向每个匹配的元素之后插入内容
      insertAfter() 向每个匹配的元素插入到指定的元素之后
      before() 向每个匹配的元素之前插入内容
      insertBefore() 向每隔匹配的元素插入到指定元素之前
    • e.g.1 
      $(function() {
  //1. 创建一个<li name="new">new-list</li>
  //2. 将其加入到ul的子节点
  $("<li name='new'>new-list</li>").appendTo($("ul"));
  //测试
  alert($("li[name='new']").text());
})
    • e.g.2 
      $(function() {
  //1. 创建一个<li name="new">new-list</li>
  //2. 将其加入到第二个li的下一个
  $("<li name='new'>new-list</li>").insertAfter($("li")[1]);
})
  5. 删除节点
    • remove() - 从DOM中删除所匹配的元素，传入的参数用于根据jQuery表达式来筛选元素， 当某个节点用remove()方法删除后，该节点所包含的所有后代节点同时被删除， 这个方法的返回值是一个指向被删除节点的引用
    • empty() - 清空节点，清空元素中的所有后代元素中的节点，不包含属性结点
  6. 克隆节点
    • clone() - 克隆匹配的DOM元素，返回值为克隆后的副本，此时新克隆的节点不具备任何行为
    • clone(true) - 克隆元素的同时也克隆元素中的事件
    • e.g. 
      $(function() {
  $($("li")[0]).on("click", function(){
    alert($(this).text());
  })
  var $li_clone = $($("li")[0]).clone(true);
  $li_clone.attr("id", "li-1-clone").appendTo($("ul"));
})
  7. 替换节点
    • node1.replaceWith(node2) - 使用node2替换node1
    • node1.replaceAll(node2) - 使用node1替换node2
    • e.g.
      $("<li>new-li</li>").replaceAll($("ul li:first"));
    • 交换两个节点 
      $(function() {
  //交换li-1和li-3两个节点
  $li1 = $("ul li:first");
  $li3 = $("ul li:eq(2)");
  //克隆li-1节点
  $li1_clone = $("ul li:first").clone(true);
  //将克隆节点替换li-3节点
  $li1_clone.replaceAll($li3);
  //将替换下来的li-3节点替换li-1节点
  $li3.replaceAll($li1);
})
  8. 包裹节点
    • wrap(): 将指定节点用其他标记包裹起来，该方法用于需要在文档中插入额外的结构化的标记非常有用，不会破坏原始文档的语义
    • wrapAll(): 将所有匹配的元素用一个元素来包裹，而wrap()方法是将所有元素进行单独的包裹
    • wrapInner(): 将每一个匹配的元素的子内容(包括文本节点),用其结构化标记包裹起来
  9. 属性操作
    • attr(): 获取属性和设置属性
      当为该方法传递一个参数时，即为某元素的获取指定属性
      为该方法传递两个参数时，即为某元素设置指定属性的值
    • removeAttr(): 删除指定元素的指定属性
  10. 设置和获取HTML，文本和值
    • html(): 读取和设置某个元素的HTML内容，该方法可用于XHTML，不可以用于XML
    • text(): 读取和设置某个元素中文本内容，该方法可以用于XHTML，也可以用于XML文档
    • val(): 读取和设置某个元素的value值，适用于文本框，下拉列表，单选框的等值

4. CSS DOM

  1. class操作
    • 获取class和设置class: class是元素的一个属性，所以获取class，何止class都可以使用attr()方法来完成
    • 追加样式： addClass()
    • 移除样式: removeClass()，从匹配元素中删除全部或指定的class
    • 切换样式: toggleClass()，控制样式上的重复切换，如果类名存在则删除它，如果类名不存在则添加它
    • 判断是否含有某个样式: hasClass()
    • e.g. 
      $(function() {
  //判断第一个li是否为.redLi，如果是则修改为.greenLi
  var $li = $("li:first");
  if ($li.hasClass("redLi")) {
    $li.removeClass("redLi");
    $li.addClass("greenLi");
  }
})
  2. CSS DOM操作
    • 获取和设置元素的样式属性 : css()
    • 获取和设置元素的透明度 : opacity属性
    • 获取和设置元素的宽度，高度：height()， width()；在设置值时，若只传递数字， 则默认单位为px，如果需要其他单位则需要传递字符串类型，e.g. $("p:first").height("2em");
    • 获取元素在当前视窗中的相对位移 : offset()，其返回对象包含两个属性：top, left，该方法只对可见元素有效
      e.g. 获取当前元素的相对位移 
      $(function() {
  var $div = $("div");
  //获取div的顶部偏移量
  var offsetTop = $div.offset().top;
  //获取div的左部偏移量
  var offsetLeft = $div.offset().left;
  alert("offset top is " + offsetTop);
  alert("offset left is " + offsetLeft);
})

5. 事件处理

  1. 加载DOM方式
    方式 原生方式 jQuery方式
    方法 必须等待网页中的所有内容加载完毕
    (包括图片)后执行    		 网页中的所有DOM结构绘制完毕后
    执行,可能DOM元素关联的东西并没有加载完
    编写个数 不能同时编写多个 可以同时编写多个
    简化方法 无 $(function() {... ...})
  2. 事件绑定
    • is() : 判断当前jQuery对象是否符合选择器选择的要求
      e.g. var flag = obj.is("li:first");
    • bind() : 为jQuery对象绑定事件
      e.g. $obj.bind("click", function() { ... ... });
  3. 合成事件
    • hover() : 模拟光标悬停事件，当光标移动到元素上时，会触发第一个函数，当光标移出这个元素时，会触发第二个函数
      e.g. $obj.hover(function() {...}, function() {...});
    • togger() : 模拟鼠标连续点击事件，第一次单击元素时，触发第一个函数，当再一次单击同一个元素时，触发第二个函数，如果有更多函数则依次触发；该方法还可以切换元素的可见状态
      e.g. $obj.toggle(function() {...}, function(){...});
  4. 事件冒泡
    • 事件会按照DOM层次结构像水泡一样不断向上止于顶端
    • 解决方案：在事件处理函数中返回false，会对事件停止冒泡，还可以停止元素的默认行为
    • $(function() {
  $(".bigDiv").click(function(){
    alert("bigDiv is onclick");
    return false;
  })
  $(".smallDiv").click(function() {
    alert("smallDiv is onclick");
    return false;
  })
})
  5. 事件对象
    • 事件对象：当触发事件时，事件对象就被创建了，在程序中使用事件只需要为函数添加一个参数，该事件对象只有事件处理函数才可以访问到，事件处理函数执行完毕后，事件对象会被销毁
    • 得到事件对象的方法：在相应函数中添加一个参数function(event) {... ...}
    • 事件对象的两个属性：event.pageX, event.pageY - 获取到光标相对于页面的x,y坐标
    • e.g. 
      $(function() {
  //鼠标移动时会触发事件，生成事件对象
  $("div").mousemove(function(event) {
  //将鼠标移动的当前页面的坐标动态显示在div中的span内
  $("span").text("x : " + event.pageX + " , y : " + event.pageY);
  });
})
  6. 移除事件
    • 移除某按钮上的所有click事件：$("btn").unbind("click")
    • 移除某按钮上的所有事件：$("btn").unbind()
    • one() : 该方法可以为元素绑定处理函数，当处理函数触发一次后，立即被删除，即在每个对象上事件处理函数只会被执行一次

6. 动画

  1. 隐藏和显示
    • hide() : 在HTML文档中，为一个元素调用hide()方法会将该元素的display样式改为none，代码功能等同css("display", "none")
    • show() : 将元素的dsiplay样式改为先前的显示状态
    • 以上两个方法在不加任何参数的情况下，作用是立即隐藏或显示匹配的元素，不会有任何动画，通过传入速度参数可以是元素在规定时间内进行变化
    • 以上两个方法会同时改变内容的高度，宽度，不透明度
  2. 淡入和淡出
    • fadeIn() : 改变元素的透明度，直到元素全部出现
    • fadeOut() : 改变元素的透明度，直到元素全部消失
  3. 滑动
    • slideDown() : 改变元素的高度，元素由上至下出现
    • slideUp() : 改变元素的高度，元素由下至上消失
  4. 切换元素可见状态
    • toggle() : 如果元素课件，则切换为隐藏；如果元素隐藏，则切换为可见
    • slideToggle() : 通过高度变化来切换匹配元素的可见性
    • fadeToggle() : 通过透明度变化切换匹配元素的可见性
    • fadeTo() : 把不透明度以渐近的方式调整到指定的值，第一个参数传入速度，第二个参数传入变化后的透明度
  5. 自定义动画
    • $(selector).animate({params},speed,callback)第一个参数为动画结束后需要达到的效果数组，第二个参数为动画速度，第三个参数为回调函数
    • e.g. 
      $("button").click(function(){
  $("div").animate({
    left:'250px',
    opacity:'0.5',
    height:'150px',
    width:'150px'
  }, 1000);
});
  6. 停止
    • stop(): 停止当前元素的动画或效果
    • $(selector).stop(stopAll,goToEnd): 第一个参数默认为false，如果true则会在停止当前动画后执行动画队列的后续动画； 第二个参数默认为false，如果true则会停止动画并且使元素立即达到动画完成后的效果
  7. 回调函数
    • 在动画参数的最后一个参数传入闭包函数，则会在动画执行完毕后立即执行闭包函数的内容
    • e.g. 
      $("btn").click(function() {
  $("div").show(1000, function() {
    $("div").hidden(1000);
  })
});
